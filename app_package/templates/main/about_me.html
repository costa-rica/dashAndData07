About me


Hi, I’m Nick! I’m a full stack developer familiar with Python web frameworks. I build web dashboard interfaces supported by databases. I manage everything from writing the code to web hosting on AWS. My expertise lies within the Flask Framework.

I'm a proud father of a daugther making her way in Manhattan, NY. I love creating, reading and sports.
